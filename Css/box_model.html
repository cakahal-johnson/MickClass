CSS Box Model

Is used to wrap Hmtl Element, which consists of margins, borders, padding and the actual content

Different parts of css Box Model

  1. Content: is where the images and text appears
  2. Padding: it clears an area around the content whereby the padding is transparent
  3. Border: it goes around the padding and the content using three params 1-size 2-typeOf 3-color
  4. Margin: it also clears an area outside the border the border, it's also transparent
