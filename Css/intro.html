Css Intro

Definition: Cascading Style Sheets

it helps to decribe how the HTML elements are to be displayed on Screen, paper , or in other media

  • it also saves time while working, it can controls the layout of mutiple web pages all at once

    • TYPES OF CSS CONNECTION

    1. Inline CSS: is a way of styling HTML within it's Elements TAGS as an Attributes calling up Style="property: value; ..."
    2. Internal Css: this is by using Style TAg placed below head closing tag above body opening tag still in the same HTML file
    3. External Css: this is handy as a stylesheet placed outside the HTML file best in a folder then we link in head section below the title using the Link TAg then inside the style sheet no Tag needed

    CSS Comment:

    Same as in HTML using crlt / but the sign is /* content */

    CSS Syntax for internal and external:

    it starts with a Selector e.g HTML Element tags, class as attributes which is selected using .dot, id as an Attributes which is selected using #, universal means all the HTML elements inside the body Tag which is selected using *

    Css Declaration: Selector { property: value; }
